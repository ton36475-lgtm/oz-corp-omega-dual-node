# OZ-CORP OMEGA Advanced Builder Skill

## Skill Name: `oz-corp-omega-builder-advanced`

## Description
This skill encapsulates the entire process of building and deploying the OZ-CORP OMEGA Dual-Node system, adhering to MillerDev standards. It leverages a Claude-Codex hybrid workflow, integrating advanced monorepo structures, AI-driven dashboards, intelligent nervous systems, multi-MCP orchestration, and AI-powered Kanban with predictive analytics. This skill enables the automated setup of a production-grade distributed computing platform from a bare machine.

## Goal
To fully provision and configure the OZ-CORP OMEGA Dual-Node system, including all four pillars, with integrated Manus API orchestration and a robust development environment, ready for deployment and operation.

## Prerequisites
- **GitHub Access:** Configured `gh` CLI with access to `ton36475-lgtm/oz-corp-omega-dual-node`.
- **pnpm:** Installed globally for monorepo management.
- **Docker & Docker Compose:** Installed and running for containerized services.
- **Python 3.11+:** With `python-dotenv` and `pymongo` installed.
- **Node.js 20+:** For Next.js and other JavaScript-based components.
- **Cloudflare Account:** For deploying the frontend dashboard.
- **MCP Credentials:** API keys for Meta Ads, Context7, DeepSeek V4, and Supabase (as defined in `.env.example`).

To invoke this skill, the agent should follow the detailed steps outlined below, executing commands sequentially in a clean sandbox environment.
